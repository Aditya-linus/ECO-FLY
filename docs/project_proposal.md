# Project Proposal

## Project Title

ECO-FLY

Environmental & Agricultural Aerial Robotics Platform

---

## WHY did I build it??

Modern agricultural and environmental monitoring methods often require manual inspection over large areas.

To overcome this I decided to make budget friendly alternative option for agriculture.

---

## Objectives

1. Build a modular FPV drone platform
2. Enable environmental monitoring applications
3. Support agricultural observation tasks
4. Document assembly and testing procedures
5. Create an open-source aerial robotics platform

---

## Expected Outcomes

- Functional FPV drone
- Technical documentation
- Test reports
