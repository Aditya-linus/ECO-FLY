# Flight Controller (FC)

Purpose:
Acts as the central computer of the drone.

Function:
- Reads sensor data
- Processes pilot commands
- Stabilizes the aircraft
- Controls motor outputs
- Executes flight algorithms
- Blah blah blah its the main chip 

---

# Electronic Speed Controller (ESC)

Purpose:
Controls motor speed.

Function:
- Receives commands from the flight controller
- Regulates power delivered to each motor
- Enables precise thrust adjustments
- The chip used to control motors lol.

---

# FPV Camera

Purpose:
Provides real-time visual feedback.

Function:
- Captures forward-facing video
- Sends live footage to the pilot
- Supports navigation and inspection tasks

---

# Video Transmitter (VTX)

Purpose:
Transmits video from the drone to the ground station and enables fpv veiw.

---

# ELRS Receiver

Purpose:
Receives control signals from the radio transmitter.

also:
- Transfers pilot commands to the flight controller
- Maintains reliable communication link
- Supports low-latency control

---

# Radio Transmitter

well a transmitter is like a remote control for your drone
- Sends flight commands
- Controls throttle, pitch, roll, and yaw
- Enables manual drone operation
